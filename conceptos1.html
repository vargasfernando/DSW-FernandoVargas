CONCEPTOS


GitHub es el medio por el cual se pueden alojar repositorios gestionados por el sistema de controles Git. GitHub es un sitio web que utiliza Git para ofrecerles a los programadores repositorios de software, este tiene como objetivo compartir código de una forma fácil y sencilla y al mismo tiempo poder darle reconocimiento al control de versiones Git.

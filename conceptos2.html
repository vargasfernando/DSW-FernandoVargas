CONCEPTOS


Git es un Sistema de controles multiplataforma en el que puedes crear repositorios locales en los sistemas operativos mas conocidos como los son: Windows, Linux o Mac. Existen múltiples interfaces de usuario grafica para poder trabajar, aprender y crear diferentes programas y páginas web.
